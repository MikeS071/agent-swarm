## Objective
Replace generic generation with deterministic prompt compilation from structured ticket fields + policy context.

## Depends

## Files to touch

- cmd/prompts_cmd.go
- internal/feature or new internal/prompts/* package

## Implementation Steps
1. Add `prompts build <ticket|--all>` command.
2. Compile prompt sections in fixed order (spec section 6).
3. Emit manifest: swarm/prompts/<ticket>.manifest.json.
4. Preserve `prompts gen` as scaffold helper (do not remove yet).
5. Ensure deterministic output (same input => same output bytes).
