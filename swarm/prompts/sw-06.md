### 3. Features
- Auto-refresh every 3 seconds (re-read tracker + check agent status)
- Progress bars from internal/progress package (read tmux output for PROGRESS: markers)
- Keybindings per SPEC.md
- `Enter` on a ticket: switch to a detail view showing live tmux output (tail -50 of pane)
- `Esc` from detail: back to list
- Color coding: green=done, yellow=running, gray=queued, red=failed, dim=blocked
- `p` to switch projects if multiple swarm.toml found
- `Tab` toggle compact (one line per ticket) vs detailed (with progress bars)

### 4. CLI Integration
- `cmd/status.go` already exists (from sw-01). Add `--watch` flag.
- When `--watch`: launch bubbletea TUI
- When no `--watch`: print table and exit (existing behavior)
- `--compact` flag for minimal one-line output
