# SW-04: Worktree Manager

## Context
You are building a Go CLI called `agent-swarm`. Read SPEC.md for full spec.

## Your Scope
Git worktree lifecycle management in `internal/worktree/`.

### 1. Worktree Package (internal/worktree/worktree.go)
```go
type Manager struct {
    RepoDir      string   // main repo path
    WorktreeBase string   // where worktrees go (default: <repo>-worktrees/)
    BaseBranch   string   // branch to create worktrees from (default: main)
}

func New(repoDir, worktreeBase, baseBranch string) *Manager
func (m *Manager) Create(ticketID, branch string) (string, error)  // returns worktree path
func (m *Manager) Remove(ticketID string) error
func (m *Manager) Prune() error                    // git worktree prune
func (m *Manager) List() ([]Worktree, error)       // list all worktrees
func (m *Manager) Exists(ticketID string) bool
func (m *Manager) Path(ticketID string) string
func (m *Manager) HasCommits(ticketID, baseBranch string) (bool, string, error)  // bool, latestSHA, error
func (m *Manager) CleanupOlderThan(duration time.Duration) ([]string, error)  // returns removed IDs
```

### 2. Implementation Details
- `Create`: `git worktree add -b <branch> <path> <baseBranch>`
- WorktreeBase default: sibling dir `<reponame>-worktrees/`
- Worktree path: `<worktreeBase>/<ticketID>/`
- `HasCommits`: `git log <baseBranch>..<branch> --oneline` in the worktree, check if non-empty
- `List`: `git worktree list --porcelain` parsed
- `CleanupOlderThan`: check worktree creation time via stat, remove if older

### 3. CLI Commands
- `cmd/cleanup.go` — `swarm cleanup [--older-than 24h]`
- `cmd/prompts.go` — `swarm prompts check` (verify prompt files exist) + `swarm prompts gen <ticket>` (template)

### 4. Prompt Management
- `prompts check`: for each todo ticket, verify `<prompt_dir>/<ticketID>.md` exists
- `prompts gen <ticket>`: generate a template prompt file with ticket desc, deps, and placeholder sections
